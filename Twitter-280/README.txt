Twitter-280 dataset is crawled using Twitter REST API in Python 2.7 by applying the hashtag-based annotation technique criteria. All tweets are based on Twitter's 280 characters limit, wherein sarcasm-based tweets are collected using #sarcasm hashtag, whereas non-sarcasm tweets are collected using the hashtags, like #not, #education, #politics, #love, and #hate. A short statistics of two files is given below.

1. Sarcasm.txt: This file contains a total of 17488 tweet ids for sarcasm category. 

2. Non_Sarcasm.txt: This file contains a total of 25134 tweet ids for non-sarcasm category.
